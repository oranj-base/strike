# How to Use the Activity Section

The Activity section on BetBTC provides users with a comprehensive overview of all their financial transactions, betting history, and rewards on the platform. Once logged in, find the "Activity" option in the main menu on the left side of the dashboard.&#x20;

This section is divided into three primary categories: **Funds**, **Bettings**, and **Rewards**.  Each category provides detailed information relevant to the user's activities on the platform.

### **1. Funds Activity**

The **Funds** tab in the Activity section records all financial transactions related to deposits, transfers, and withdrawals. This section provides the following details for each transaction:

<figure><img src="../.gitbook/assets/image.png" alt=""><figcaption><p>Fig1: Funds Activity</p></figcaption></figure>

| Data        | Description                                                 |
| ----------- | ----------------------------------------------------------- |
| **Date**    | The date and time when the transaction was completed.       |
| **Types**   | The type of transaction (Deposit, Transfer, or Withdrawal). |
| **Amount**  | The total amount involved in the transaction.               |
| **Address** | The wallet address associated with the transaction.         |

### **2. Bettings Activity**

The **Bettings** tab is where users can view the history of all bets placed on the platform. This section includes:

<figure><img src="../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

| Data                    | Description                                               |
| ----------------------- | --------------------------------------------------------- |
| **Date**                | The date when the bet was placed.                         |
| **Market ID**           | The unique identifier for the betting market.             |
| **Option ID**           | The specific option (Yes, No, etc.) that the user bet on. |
| **Types**               | The type of bet placed (Single, Multiple, etc.).          |
| **Token Amount**        | The number of tokens wagered in the bet.                  |
| **Token Price**         | The price per token at the time the bet was placed        |
| **Total Amount in USD** | The total value of the bet in USD.                        |
| **Total Amount in BTC** | The total value of the bet in BTC.                        |

***

### **3. Rewards Activity**

The **Rewards** tab tracks all the points and rewards earned through various activities on the platform, such as referrals, betting milestones, and more. This section includes:

<figure><img src="../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

| Data      | Description                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| **Date**  | The date when the reward was credited.                                            |
| **Types** | The type of reward or bonus earned (Referral Bonus, Milestone Achievement, etc.). |
